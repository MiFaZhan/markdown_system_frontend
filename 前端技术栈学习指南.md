# 前端技术栈学习指南

## 项目概述

这是一个基于 Vue 3 的 Markdown 文档管理系统前端项目，主要功能包括项目管理、文件树展示、Markdown 编辑和大纲导航。

## 核心技术栈

### 1. Vue 3 (核心框架)
- **版本**: 3.3.4
- **特点**: 组合式 API (Composition API)
- **学习重点**:
  - `<script setup>` 语法糖
  - `ref()` 和 `reactive()` 响应式数据
  - `computed()` 计算属性
  - `watch()` 监听器
  - 生命周期钩子 (`onMounted`, `onBeforeUnmount`)

### 2. Vue Router 4 (路由管理)
- **版本**: 4.2.4
- **功能**: 单页应用路由
- **学习重点**:
  - 路由配置和嵌套路由
  - 路由守卫 (`beforeEach`)
  - 动态路由参数
  - 编程式导航

### 3. Pinia (状态管理)
- **版本**: 2.1.6
- **替代**: Vuex 的现代化替代品
- **学习重点**:
  - Store 定义和使用
  - 状态、计算属性、动作
  - 组合式 Store

### 4. Element Plus (UI 组件库)
- **版本**: 2.3.9
- **特点**: 基于 Vue 3 的企业级 UI 库
- **学习重点**:
  - 常用组件 (Button, Input, Tree, Dropdown 等)
  - 图标系统
  - 主题定制
  - 国际化配置

### 5. Vite (构建工具)
- **版本**: 4.4.5
- **特点**: 快速的前端构建工具
- **学习重点**:
  - 开发服务器
  - 热模块替换 (HMR)
  - 插件系统
  - 构建优化

### 6. Vditor (Markdown 编辑器)
- **版本**: 3.11.2
- **功能**: 所见即所得的 Markdown 编辑器
- **学习重点**:
  - 编辑器配置
  - 内容获取和设置
  - 事件处理

## 快速学习路径

### 第一阶段：基础概念 (1-2 周)

#### 1. JavaScript ES6+ 基础
- 箭头函数、解构赋值、模板字符串
- Promise、async/await
- 模块化 (import/export)

#### 2. Vue 3 核心概念
**推荐学习资源**:
- [Vue 3 官方文档](https://cn.vuejs.org/)
- [Vue 3 组合式 API 指南](https://cn.vuejs.org/guide/extras/composition-api-faq.html)

**重点掌握**:
```javascript
// 组合式 API 基本用法
import { ref, computed, watch, onMounted } from 'vue'

export default {
  setup() {
    // 响应式数据
    const count = ref(0)
    
    // 计算属性
    const doubleCount = computed(() => count.value * 2)
    
    // 监听器
    watch(count, (newVal) => {
      console.log('count changed:', newVal)
    })
    
    // 生命周期
    onMounted(() => {
      console.log('组件已挂载')
    })
    
    return { count, doubleCount }
  }
}
```

#### 3. 单文件组件 (SFC)
理解 `.vue` 文件的三个部分：
- `<template>`: 模板
- `<script setup>`: 逻辑
- `<style scoped>`: 样式

## 常见问题解答

### Q: 为什么使用组合式 API 而不是选项式 API？
A: 组合式 API 提供更好的逻辑复用和 TypeScript 支持，代码组织更灵活。

### Q: Pinia 相比 Vuex 有什么优势？
A: Pinia 更轻量、API 更简洁、对 TypeScript 支持更好、没有 mutations 概念。

### Q: 如何快速定位问题？
A: 
1. 查看浏览器控制台错误信息
2. 使用 Vue DevTools 检查组件状态
3. 在关键位置添加 console.log
4. 检查网络请求是否正常

### Q: 如何学习 CSS 样式？
A: 
1. 理解 CSS 基础：选择器、盒模型、定位
2. 学习 Flexbox 和 Grid 布局
3. 掌握响应式设计原理
4. 了解 CSS 变量和预处理器

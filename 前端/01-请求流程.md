## 1. 模板字符串 (Template Literals)

### 什么是模板字符串

模板字符串是 JavaScript 中的一种字符串表示方法，使用**反引号** `` ` `` （而不是单引号 `' ` 或双引号 `"`）包裹。

### 基本语法

```javascript
const name = 'Alice'
const greeting = `Hello, ${name}!`  // 结果: "Hello, Alice!"
```

### 在项目中的应用

在 [contentService.js](file:///D:\Project\markdown_system\markdown_system_frontend\src\api\contentService.js) 中：

```javascript
export function getMarkdownContent(nodeId) {
  return get(`/markdown-content/${nodeId}`)
}
```

这里的 `` `/markdown-content/${nodeId}` `` 会在运行时将 `nodeId` 的值嵌入到字符串中，生成类似 `/markdown-content/123` 这样的 URL 路径。

### 对比

| 方式 | 写法 | 结果（假设 nodeId = 123） |
|------|------|---------------------------|
| 传统拼接 | `"/markdown-content/" + nodeId` | `"/markdown-content/123"` |
| 模板字符串 | `` `/markdown-content/${nodeId}` `` | `"/markdown-content/123"` |

## 2. Promise 与异步请求

### Promise 概念

`Promise` 是 JavaScript 中表示异步操作最终完成（或失败）的对象。

```javascript
// getMarkdownContent 返回的是一个 Promise
export function getMarkdownContent(nodeId) {
  return get(`/markdown-content/${nodeId}`)  // 返回 Promise
}
```

### 调用方式

调用者使用 `await` 或 `.then()` 来获取 Promise 的结果：

```javascript
// 方式1: async/await
const contentData = await getMarkdownContent(fileId)

// 方式2: .then()
getMarkdownContent(fileId).then(contentData => {
  // 处理返回的数据
})
```

## 3. HTTP 请求封装

### 请求流程

项目中的请求流程分为多层：

### 各层职责

| 层级 | 文件 | 职责 |
|------|------|------|
| 业务函数 | contentService.js | 定义具体 API 端点 |
| 封装函数 | request.js - get() | 处理 URL 参数拼接 |
| 通用请求 | request.js - request() | 处理认证、错误处理、响应解析 |
| 浏览器 API | fetch | 实际发送网络请求 |

### 关键代码

**request.js 中的 get 函数：**

```javascript
export function get(url, params = {}) {
  const queryString = new URLSearchParams(params).toString()
  const fullUrl = queryString ? `${url}?${queryString}` : url
  return request(fullUrl)
}
```

**request.js 中的 request 函数：**

```javascript
async function request(url, options = {}) {
  const token = localStorage.getItem('token')
  
  const headers = {}
  if (token) {
    headers['Authorization'] = `Bearer ${token}`
  }
  
  const fullUrl = `${BASE_URL}${url}`
  
  const response = await fetch(fullUrl, {
    ...options,
    headers: { ...headers, ...options.headers }
  })
  
  const data = await response.json()
  return data.data
}
```

## 5. nodeId 的来源

`nodeId`（也称为 `fileId`）是文件的唯一标识符：

- **来源**：用户在文件树中点击的文件对象
- **获取方式**：从 `file.id` 属性获取
- **用途**：作为参数传递给后端 API，用于获取对应文件的 Markdown 内容

```javascript
// Workspace.vue
const openFile = async (file) => {
  const contentData = await loadFileContent(file.id)  // ← file.id 就是 nodeId
}
```
